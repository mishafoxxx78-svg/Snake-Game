 🐍 Snake Game (Python + Pygame)

📌 О проекте

Классическая игра "Змейка", реализованная на Python с использованием библиотеки Pygame.

Проект создан как pet-project для практики:

* работы с игровыми циклами
* обработки событий
* базовой графики



 🚀 Возможности

* 🎮 Классический геймплей Snake
* 🍎 Генерация еды в случайных местах
* 💥 Проверка столкновений (стены и тело)
* 📊 Подсчёт очков

---

 🛠️ Технологии

* Python 3
* Pygame

---

## ⚙️ Установка и запуск

```bash
git clone https://github.com/your-username/snake-game.git
cd snake-game
pip install pygame
python snake_game.py
```

---
 🎮 Управление

| Клавиша | Действие |
| ------- | -------- |
| ↑       | Вверх    |
| ↓       | Вниз     |
| ←       | Влево    |
| →       | Вправо   |

---

🧠 Как работает игра

1. Игровой цикл постоянно обновляет экран
2. Игрок управляет направлением змейки
3. Змейка двигается и растёт при поедании еды
4. Игра заканчивается при столкновении
