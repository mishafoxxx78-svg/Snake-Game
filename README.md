
 🐍 Игра "Змейка" (Python + Pygame)

 📌 Описание
Это простая игра "Змейка", созданная с использованием Python и Pygame.

## 🚀 Особенности
- Классический геймплей "Змейки"
- Отслеживание очков
- Обнаружение столкновений

 🛠️ Установка
```bash
git clone https://github.com/your-username/snake-game.git
cd snake-game
pip install pygame
python snake_game.py
```
